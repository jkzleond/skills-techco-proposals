# 优秀案例参考

本目录存放优秀的售前方案案例，供学习参考。

## 案例位置

项目根目录的 `售前/` 文件夹包含以下优秀案例：

1. **会员积分商城建设方案.md**
   - 完整的四大板块结构
   - 图文并茂，使用 ASCII 图和表格
   - 流畅的段落式叙述

2. **官网及业务系统升级方案.md**
   - 清晰的系统架构说明
   - 详细的集成关系描述
   - 商务化的语言风格

## 查找方法

从项目根目录 `skills-techco-proposals/` 进入 `售前/` 文件夹：

```bash
cd skills-techco-proposals/售前/
ls -la
```

## 学习要点

参考这些案例时，注意观察：

1. **结构清晰**：四大板块完整，层级明确
2. **图文并茂**：大量使用 ASCII 图、表格、流程图
3. **内容流畅**：段落式叙述，避免清单式
4. **引用反馈**：使用会员反馈增强说服力
5. **具体场景**：用具体的使用场景和示例
6. **无技术细节**：不包含技术架构、数据库设计等
7. **语言商务**：简洁明了，专业商务
